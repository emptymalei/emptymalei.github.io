Toolkit


Here is a list of tools hosted on our server. Some of them are just a fork of their original. The purpose of them, is to make it more convinient for me to use them in the edu network.
Other pages at http://multiverse.lamost.org/testbed/Other
Show:
NGUniverseMap: A illustration of the structure of the universe.
SolvayConference1927: The famous Solvay Conference in 1927. Zooming function implanted.
WhereAreWe: Showing where we are in the universe.
Management:
StickyNotes: As the name implies, Sticky Notes is a sticky note tool developed using the local storage function of HTML5. A stand alone version of a chrome app.
polarclock: A beautiful clock developed using HTML5.
lambda: A open source web list page.
Writing:
graph.tk: A graphing tool can be used to plot many functions. Pretty design. Check the source at github.
jaxedit: A online LaTeX editor.
matheditor: A revised JaxEdit.
MISC:
about: A simple CV template.
gcgroup: A web page for research teams. This is modified for our group, but never used.
googleguitar: The famous google guitar doodle.
historyguide1: Historical guide page for Astronomy Association of Shandong University website. Draft state.
