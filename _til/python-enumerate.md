`enumerate()` can be used to keep track of the index of the list.

{% highlight python %}
test = [None] * 5
for n, elem in enumerate( range(5,10) ):
    print n, elem
    test[n] = elem*2
{% endhighlight %}

Here we also used the technique of creating a list of length 5.
